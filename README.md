Next Pizza 🍕
A full-featured pizza delivery web application inspired by Dodo Pizza, built with modern web technologies.

📱 Overview
Next Pizza is a complete pizza ordering platform that allows users to browse products, customize pizzas, add items to cart, and place orders with delivery address. The application features authentication, product filtering, story highlights, and a seamless checkout process.

✨ Features
Product Catalog - Browse pizzas, snacks, drinks, and other items

Product Customization - Choose pizza size, crust type, and ingredients

Shopping Cart - Add, remove, and modify items in cart

User Authentication - Register/login via credentials or GitHub

Address Autocomplete - Russian address suggestions with DaData API

Order Management - Place orders and track status

Stories - Instagram-like story highlights for promotions

Email Notifications - Order confirmations via Resend

Admin Dashboard - Manage products and orders (planned)

Responsive Design - Mobile-friendly interface

🛠 Technology Stack
Core Framework
Next.js 15 (React 19 RC) - Full-stack React framework with App Router

TypeScript - Type safety and better developer experience

Database & ORM
PostgreSQL - Primary database

Prisma - Type-safe ORM with migrations

Authentication
NextAuth.js (Auth.js) - Authentication with credentials and GitHub OAuth

bcrypt - Password hashing

Styling & UI
Tailwind CSS - Utility-first CSS framework

Radix UI Primitives - Accessible component primitives

class-variance-authority - Component variants management

lucide-react - Icon library

vaul - Drawer component for mobile

react-insta-stories - Instagram-like stories

Forms & Validation
React Hook Form - Performant form management

Zod - Schema validation

@hookform/resolvers - Zod integration with React Hook Form

State Management
Zustand - Lightweight state management (cart, modals)

React Hooks - Local state and effects

API & HTTP
Axios - HTTP client for API requests

qs - Query string parsing/stringifying

Utilities
react-use - Collection of essential React hooks

@reactuses/core - Additional React utilities

react-dadata - Russian address suggestions

clsx + tailwind-merge - Conditional class name management

Email
Resend - Email service for order confirmations

Development Tools
ESLint - Code linting

PostCSS - CSS processing
